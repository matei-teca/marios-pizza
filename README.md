# Welcome to Mario's Pizza Website!

A website for ordering pizza!
This is a study team project at Codecool.
Created as a team with Vali and Marian

### Install dependencies
(for Bootstrap, Express, Nodemon)
```bash
cd ./server
npm install
```

### Running the code
```bash
cd ./server
node server.js
```
